### Task 05-02: Add handleNameOwnerChanged slot to DaemonInterface

**Action:**
1. In `gui/src/daemon_interface.h`, add a `private slots:` section after `public slots:` with:
```cpp
private slots:
    void handleNameOwnerChanged(const QString &name, const QString &oldOwner, const QString &newOwner);
```

2. In `gui/src/daemon_interface.cpp`, add the implementation after `setLastError()`:
```cpp
void DaemonInterface::handleNameOwnerChanged(const QString &name, const QString &oldOwner, const QString &newOwner)
{
    Q_UNUSED(oldOwner);
    if (name == QLatin1String(s_service)) {
        if (newOwner.isEmpty()) {
            setConnected(false);
        } else if (oldOwner.isEmpty()) {
            setConnected(true);
        } else {
            // Daemon restarted — disconnect then reconnect
            setConnected(false);
            setConnected(true);
        }
    }
}
```
